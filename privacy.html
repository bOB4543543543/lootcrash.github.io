Privacy Policy

1. Overview

This Privacy Policy explains what LootCrash stores when you use the Discord bot. LootCrash is designed as a Discord-only game bot and does not require a website dashboard or server admin configuration panel.

2. Data We Store

LootCrash may store the following data linked to your Discord user ID:

  • Wallet, bank, net worth, and economy progress.
  • XP, level, badges, command usage stats, and cooldown timestamps.
  • Inventory items, pet data, event votes, and heist participation.
  • Trade, payment, game, and command information needed to run bot features.

3. Data We Do Not Intentionally Store

LootCrash does not intentionally store message history, private messages, email addresses, real names, payment details, or server configuration settings.

4. How Data Is Used

Stored data is used to provide gameplay features, including balances, cooldowns, items, pets, leaderboards, profiles, stats, events, and rewards.

5. Sharing

LootCrash does not sell user data. Some gameplay information may be shown publicly in Discord when you use public commands, such as balances, leaderboards, game results, trades, heists, or social commands.

6. Data Deletion

You can delete your stored LootCrash user data with the /wipe_data command. The command opens a user-only confirmation flow before deletion.

7. Security

Reasonable efforts are made to protect the bot's data store. No online service can guarantee perfect security or permanent availability.

8. Changes

This Privacy Policy may be updated as LootCrash changes. Continued use of LootCrash after updates means you accept the updated policy.

9. Contact

For privacy questions or deletion issues, contact the bot owner through the Discord server or GitHub repository where LootCrash is distributed.
